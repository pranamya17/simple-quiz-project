# simple-quiz-project
This project is a Simple Quiz Application developed using HTML, CSS, and Vanilla JavaScript. It is designed to provide an interactive and user-friendly way to attempt multiple-choice questions while receiving instant feedback on answers.

The application dynamically loads questions from a predefined JavaScript array and displays them one at a time. Each question includes multiple answer options, where only one is correct. When a user selects an answer, the app immediately highlights whether the selected option is correct or incorrect using color indicators (green for correct and red for wrong). It also reveals the correct answer for better understanding.

The quiz maintains a running score, incrementing it for every correct answer. Users can navigate through questions using the Next button, and once all questions are completed, the application displays the final score. Additionally, users are given the option to restart the quiz and try again, making the app reusable and engaging.

The project demonstrates core concepts of front-end web development, including:

DOM manipulation for dynamically updating content
Event handling for user interactions
Use of arrays and objects to manage structured data
Conditional logic for answer validation
Dynamic creation and removal of HTML elements

The user interface is styled using CSS to ensure a clean, responsive, and visually appealing layout. The design adapts to different screen sizes, making it accessible across devices.
